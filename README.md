# CS434 Final Project: Foggy Flooding
Created by: Linda Zou, Oliver Girard, and Tongwei Zhang.

## Running the Program
To run this application using a .exe file, please run the "CS434 Final Project.exe" file located in the "x64\Release" folder.

## Libraries and Includes
If you intend to run this application using the "CS434 Final Project.sln" file, all relevant libraries are in the "libs" folder, and all relevant files that need to be included are in the "include" folder.
